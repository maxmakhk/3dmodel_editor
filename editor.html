How to Use

  • Load a .glb from model/ folder or drag-and-drop — GLBs are auto-uploaded to model/.
  • Hover a node in the list to highlight it in the 3D view. Click to select.
  • Each node can hold multiple accessories — click one to edit its transform.
  • Use 💾 Save Session to save your setup. Sessions remember file paths so everything reloads automatically.
  • When done, Export GLB downloads the model with all accessories baked in.
